# Estructura del repositorio

```text
code/          Codigo fuente de la aplicacion PHP
code/css/      Hojas de estilo organizadas por zonas
code/js/       JavaScript de interaccion y escaparate React
code/img/      Imagenes usadas por la tienda
code/templates/Plantillas reutilizables PHP
docs/          Documentacion tecnica breve
screenshots/   Capturas usadas en el README
```

La carpeta `code/` equivale a la raiz publica de la web. En un hosting tradicional, su contenido podria subirse a `htdocs`, `public_html` o equivalente.
